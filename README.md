# Quiz-App

Dieses Projekt ist eine interaktive Quiz-App.

## Beschreibung
Die Quiz-App stellt dem Benutzer Fragen, die entweder richtig oder falsch
beantwortet werden können. Nach jeder Antwort bekommt der Benutzer sofort
eine Rückmeldung.

- Richtige Antwort → positives Feedback + Sound
- Falsche Antwort → negatives Feedback + Sound
- Anzeige des Fortschritts (z. B. Prozent oder Status)
- Interaktive und einfache Bedienung

## Funktionen
- Multiple-Choice-Fragen
- Soundeffekte bei richtigen und falschen Antworten
- Anzeige, ob die Antwort korrekt ist
- Fortschrittsanzeige während des Quiz
- Responsive Design für Desktop und Mobile

## Technologien
- HTML
- CSS
- JavaScript
